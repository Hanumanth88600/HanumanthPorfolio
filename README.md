# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/Hanumanth88600/portfolio.git
```

Move into project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
